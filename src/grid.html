dosLotosGrid

Qué es dosLotosGrid?

Es un framework CSS basado en Flexbox que te permite crear estructuras con gran rapidez. dosLotosGrid cuenta con los componentes indispensable para poder crear un sitio web.

dosLotosGrid esta testeado corss-browser y multiplataforma. Es compatible con todos los sistemas operativos y su grilla principal es totalmente adpatable para desarrollos de aplicaciones mobiles en React.js u otros lenguajes.

Por qué cree dosLotosGrid?

Por años use Bootstrap desde su primera versión hasta su ultimo release y realmente me parece un framework excelente pero siempre pense que se podian agregar componentes y/o comportamientos para poder tener un abanico mayor a la hora de comenzar un desarrollo.

Muchas cosas rescate de Bootstrap a la hora de crear dosLotosGrid por ejemplo la nomenclkatura se sus clases para las distintas resoluciones, la grilla de 12 columnas y las alineaciones de las mismas que combinado con Flexbox crean un infinito universo de posibles estructuras.

dosLotosGrid se creó con el objetivo de darle la tranquilidad al desarrollador que no va a tener que pisar o editar CSS de más, no tiene que apegarse 100% a un framewrok para crear un sitio y mucho menos tener problemas de compatibilidad entre el framework y el plugin que desee instalar.

En pocas palabras, dosLotosGrid es el paso inicial, lo complejo, lo 'dificil de hacer'. Una vez instalado solo hay que 'maquillarlo'.

Quiénes Pueden utilizarlo?

Al igual que todos los Frameworks en CSS cualquiera puede utilizar dosLotosGrid siempre y cuando posean un grado básico de HTML/CSS.
