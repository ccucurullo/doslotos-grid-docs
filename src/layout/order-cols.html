Orden Columnas

Permite ordenar nuestras columnas independientemente del orden que presenten en HTML. Utilizando la clase 'col-(breakpoint)-ord-#'. Esta clase se puede combinar con otras para lograr layouts diferentes en distintas resoluciones 'col-lg-ord-2 col-md-ord-1'

MD-Ord-2 / LG-ord-2

- Order 1 en el HTML -
MD-Ord-3 / LG-ord-3

- Order 2 en el HTML -
MD-Ord-1 / LG-ord-4

- Order 3 en el HTML -
MD-Ord-4 / LG-ord-1

- Order 4 en el HTML -
<div class="row">

  <div class="col-sm-4 col-md-ord-2 col-lg-ord-2">...</div>

  <div class="col-sm-4 col-md-ord-3 col-lg-ord-3">...</div>

  <div class="col-sm-4 col-md-ord-1 col-lg-ord-4">...</div>

</div>
