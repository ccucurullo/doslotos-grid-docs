Auto Grid

Crea una grilla perfecta sin importarle la cantidad de elementos que contenga. Podemos setearle una cantidad de elementos por defecto y luego variar esta cantidad dependiendo de la resolución en la que se muestran.

  • grid-6 Ubica 6 elementos por línea
  • grid-md-4 Ubica 4 elementos por línea a partir de 992px
  • grid-sm-3 Ubica 3 elementos por línea a partir de 768px
  • grid-xs-2 Ubica 2 elementos por línea a partir de 480px
