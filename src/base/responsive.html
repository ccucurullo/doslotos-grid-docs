Breakpoints

Los Breakpoint o quiebres responsive estan seteados por defecto en el archivo _variables.scss pueden editarse las medidas a gusto y/o agregar las necesarias.

  • xxs -- Siempre 'flotan'
  • xs -- Breakpoint/Quiebre 480px
  • sm -- Breakpoint/Quiebre 768px
  • md -- Breakpoint/Quiebre 992px
  • lg -- Breakpoint/Quiebre 1200px

Las variables utilizadas para cada breakpoint o quiebre son las siguientes:

  • $device-xs 480px
  • $device-sm 768px
  • $device-md 992px
  • $device-lg 1200px
  • $navbar-collapse $device-sm (768px)

Nota: No es recomendable editar la variable que contiene la medida de breakpoint o quiebre. De no sentirse c√≥modos con esta variable '$device-(breakpoint)' puede crear la propia de esta manera editando el archivo _variables.scss.

@mi_nueva_variable: @device-lg default!;
